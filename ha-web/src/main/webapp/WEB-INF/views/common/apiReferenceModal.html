API 健康情報照会

登録された健康情報の照会を行う

以下の項目を必須として、リクエストを実施

リクエストURL:/ha-api/healthInfoReference

リクエストパラメータ 詳細
リクエストタイプ
/requestId		 健康情報照会のリクエスト種別
"002"を固定とする
データID
/dataId		 健康情報を登録したときに採番される健康情報を一意に識別するID
ユーザID
/userId		 健康情報アプリでユーザを一意に識別するID

レスポンスパラメータ 詳細
結果区分
/result		 API実施時の結果区分
データID
/dataId		 健康情報を登録したときに採番される健康情報を一意に識別するID
身長
/height		 データIDに紐づく健康情報に登録された身長
体重
/weight		 データIDに紐づく健康情報に登録された体重
BMI
/bmi		 データIDに紐づく健康情報に登録されたBMI
標準体重/standardWeight データIDに紐づく健康情報に登録された標準体重
登録日時/regDate データIDに紐づく健康情報に登録された登録日時
