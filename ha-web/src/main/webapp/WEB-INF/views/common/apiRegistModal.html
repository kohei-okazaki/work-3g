API 健康情報登録

指定されたリクエストパラメータからユーザの健康情報を登録する

リクエストURL:/ha-api/healthInfoRegist

リクエストパラメータ 詳細
リクエストタイプ
/requestId		 健康情報登録のリクエスト種別
"001"を固定とする
ユーザID
/userId		 健康情報アプリでユーザを一意に識別するID
身長
/height		 ユーザの登録したい身長
体重
/weight		 ユーザの登録したい体重

レスポンスパラメータ 詳細
結果区分
/result		 API実施時の結果区分
データID
/dataId		 健康情報を登録したときに採番される健康情報を一意に識別するID
ユーザID
/userId		 健康情報アプリでユーザを一意に識別するID
身長
/height		 データIDに紐づく健康情報に登録された身長(入力値)
体重
/weight		 データIDに紐づく健康情報に登録された体重(入力値)
BMI
/bmi		 データIDに紐づく健康情報に登録されたBMI
標準体重
/standardWeight		 データIDに紐づく健康情報に登録された標準体重
ユーザステータス/userStatus 前回登録された健康情報と比較し
減少した場合:10
変化なしの場合:20
増加した場合:30
登録日時
/regDate		 データIDに紐づく健康情報に登録された登録日時
