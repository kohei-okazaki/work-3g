# ha-dashboard

## 概要
健康管理アプリのダッシュボードを管理するプロジェクト

## 画面一覧

- ログイン画面
- アカウント作成画面
- アカウント設定変更画面
- パスワード再設定画面
- 健康情報登録画面
- 健康情報一括登録画面
- 健康情報照会画面
- 健康情報詳細画面
- 肺活量計算画面
- カロリー計算画面
- お知らせ画面

## ログ
- 文字コード：UTF-8
- 配置先：
    - local環境：C:/app/logs/
    - EC2環境：/var/log/app/
    - ファイル名：dashboard-${log_level}.log
