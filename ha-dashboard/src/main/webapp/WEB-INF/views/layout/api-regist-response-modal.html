健康情報登録APIレスポンスについて
レスポンスパラメータ 詳細
結果区分 result API実施時の結果区分
ユーザID userId 健康管理アプリでユーザを一意に識別するID
健康情報ID seqHealthInfoId 健康情報を登録したときに採番される健康情報を一意に識別するID
身長 height 健康情報IDに紐づく健康情報に登録された身長(入力値)
体重 weight 健康情報IDに紐づく健康情報に登録された体重(入力値)
BMI bmi 健康情報IDに紐づく健康情報に登録されたBMI
標準体重 standardWeight 健康情報IDに紐づく健康情報に登録された標準体重
健康情報ステータス healthInfoStatus 前回登録された健康情報と比較した値
  • 減少した場合:10
  • 変化なしの場合:20
  • 増加した場合:30
    		•
    健康情報作成日時 healthInfoRegDate 健康情報が登録された日時
