健康情報照会APIレスポンスについて
レスポンスボディ 詳細
result 結果区分 API実施時の結果
  • 正常終了の場合:0
  • 異常終了の場合:1
    		•
    error エラー情報
    code エラーコード エラーが発生した場合、エラーコード
    detail エラー詳細 エラーが発生した場合、詳細情報
    account アカウント情報
    seqUserId ユーザID 健康情報を登録したユーザID
    healthinfo 健康情報
    seqHealthInfoId 健康情報ID 健康情報を登録したときに採番される健康情報を一意に識別するID
    height 身長 健康情報IDに紐づく健康情報に登録された身長
    weight 体重 健康情報IDに紐づく健康情報に登録された体重
    bmi BMI 健康情報IDに紐づく健康情報に登録されたBMI
    standardWeight 標準体重 健康情報IDに紐づく健康情報に登録された標準体重
    status 健康情報ステータス 前回登録された健康情報と比較した値
  • 減少した場合:10
  • 変化なしの場合:20
  • 増加した場合:30
    		•
    regDate 健康情報作成日時 健康情報が登録された日時
